This project uses the following third party software/functions/modules.

The OS
  • Firebrick, available at http://digitalfire.ucd.ie/?page_id=1011, based on:

    • The BIOS is based on Coreboot, available at http://www.coreboot.org/Welcome_to_coreboot

The web-interface
  • jQuery, available at http://jquery.com/
  • Spinner, available at http://fgnass.github.io/spin.js/

Shell scripting
  • Translate function __, available at http://docs.unity-linux.org/I18n_Support_and_Shell_Scripts
  • sed oneliners, available at http://sed.sourceforge.net/sed1line.txt
  • awk oneliners, available at http://www.pement.org/awk/awk1line.txt
